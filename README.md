# dinos

A DNS Client

## Usage

**Usage:** _dinos [\<host>\]_

**Options:**

| Option | Description |
| -------------------- | ----------------------- |
| -V, --version | output the version number |
| -m, --max \<number\> | limit output |
| -6, --IPv6 | use IPv6 |
| -h, --help | output usage information |

## Development
`yarn build` will create the _out.js_, then you can use _./index.js_
