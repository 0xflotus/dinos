# dinos

A DNS Client

## Usage

**Usage:** _dinos [\<host>\]_

**Options:**

| Option | Description |
| -------------------- | ----------------------- |
| -V, --version | output the version number |
| -m, --max \<number\> | Limit output |
| -6, --IPv6 | Use IPv6 |
| -h, --help | output usage information |
